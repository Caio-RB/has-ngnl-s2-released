# NGNL Season 2 Tracker

A simple web project that answers one important question:

Has the second season of "No Game No Life" been released yet?

## About

This project displays:

* A clear YES or NO answer
* A live counter showing how much time has passed since the end of Season 1
* Styled using a color palette inspired by the anime

## Counter

The timer tracks:

* Years
* Months
* Weeks
* Days
* Hours
* Minutes
* Seconds

Starting from: June 25, 2014

## Technologies
